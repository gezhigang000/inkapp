# 质取report 核心竞争力与场景分析

## 核心竞争力

这个项目的壁垒不是某个单一技术点，而是一条完整的、经过打磨的**内容生产流水线**。

### 1. Prompt 工程体系（最核心）

Prompt 不是简单的"帮我写篇文章"，而是一套**多维度控制系统**：

- **人格设定**：十年经验的工程师视角，不是 AI 味的新闻播报员
- **多样性引擎**：话题池 × 视角池 × 体裁池 × 公司池 × 配色池，用日期种子做确定性随机，保证每天不重复但可复现
- **输出约束**：精确控制 HTML 结构、排版规范、内联 CSS，直接对接微信编辑器——这些是反复试错调出来的

大多数 AI 内容项目的 prompt 只有一两句话，产出千篇一律。这套 prompt 体系产出的内容有**视角差异**，这是多数项目做不到的。

### 2. 搜索 → 分析 → 排版 一体化

通过 `claude --allowedTools WebSearch,WebFetch` 让 Claude 自己搜索、自己阅读原文、自己撰写——**不是基于旧知识生成，而是实时调研**。解决了 AI 内容最大的问题：信息过时。

### 3. 发布流水线自动化

从生成到发布，完整链路：HTML 提取 → 图片处理 → 长文自动拆分 → 封面图生成 → Ink 平台上传 → 微信草稿/发布。已处理微信平台的各种限制（外链清理、标题截断、图片上传 CDN）。

### 4. 质量兜底机制

多层校验保障输出质量：

- 检测 Claude 是否返回了总结而非文章
- HTML 提取失败的 fallback
- 长文按 PART 边界智能拆分
- 图片下载失败时 DALL-E 兜底生成

### 可复用的核心能力一句话总结

> 用结构化 Prompt 控制 AI 做「实时调研 + 专业写作 + 出版级排版」，并自动化分发到多个平台。

## 可拓展场景

### 场景 1：垂直行业研报

| 维度 | 当前（AI 日报） | 新场景 |
|------|-----------------|--------|
| 话题池 | AI 方向 | 新能源/医药/芯片/加密货币 |
| 信息源 | AI 公司 | 上市公司公告、行业政策、专利数据 |
| 输出 | 微信文章 | PDF 研报 / 邮件周报 |
| 客户 | 公众号读者 | 投资机构、企业决策层 |

一份人工研报的市场价 5,000-50,000 元，流水线可以做到日更级别产出。

### 场景 2：竞品监控报告

把公司池换成客户的竞争对手列表，每天自动搜索竞品动态（产品更新、招聘、融资、专利），生成竞品周报。SaaS 公司和战略咨询的刚需。

### 场景 3：技术文档 / Changelog 生成

信息源从网络搜索换成 GitHub repo 的 commit log / PR / release notes，自动生成面向用户的 changelog 或技术文档。开源项目和 SaaS 产品都需要。

### 场景 4：多语言内容矩阵

同一套调研结果，加一层翻译 + 本地化 prompt，同时产出英文 newsletter、日文 note、中文公众号。一次调研，多平台分发。

### 场景 5：企业内部知识简报

信息源换成企业内部 wiki / Confluence / Slack 频道，每周自动整理成团队简报。大公司信息流通的真实痛点。

### 场景 6：YouTube / 播客内容二创

已有 `video_analyzer.py` 基础。可独立成产品：输入视频链接，输出图文深度解析。内容创作者和学习者都有需求。

## 产品化优先级

从投入产出比排序：

1. **垂直研报** — 离钱最近，客单价高，换一套话题池和 prompt 就能跑
2. **竞品监控** — 企业刚需，可做 SaaS 订阅模式
3. **视频二创** — 已有基础，市场需求明确

核心工作不是重写代码，而是**把 prompt 体系模板化**——让不同行业的用户能配置自己的话题池、视角池、信息源，复用同一条生产流水线。
